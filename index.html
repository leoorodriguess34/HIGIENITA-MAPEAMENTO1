Modo operacao - para usar no celular
O operador bipa o QR Code do palete, depois o codigo de barras do produto, informa a quantidade e salva. Tudo sincronizado em tempo real.
Como funciona
1. Clique em Iniciar operacao
2. Aponte a camera para o QR Code do palete (ex: A01)
3. Aponte para o codigo de barras do produto
4. Informe a quantidade
5. Repita para mais produtos no mesmo palete
6. Clique em Salvar palete
